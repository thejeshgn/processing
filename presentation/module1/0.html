What is Processing?

  • Next >

Processing is an open source programming language and integrated development environment for artists. It is used by artists to

  • create art
  • visualizations
  • simulations

This course serves as an introduction to programming in processing. Focus is on the application than theory. You start with one line of code, then add another line, run it, modify it, run it again until you get what you want. Hence we call it sketching than programming.

Note: Whenever possible a working example will be embeded in this presentation. Similar to Game of Life simulation on the index page.
